# Carbon Emission Tracker

## Overview
This web application helps users track daily activities that contribute to carbon emissions.

## Features
- Activity logging
- Emission calculation
- Dashboard with charts
- User login & registration
- Personalized tips

## Tech Stack
- HTML, CSS, JavaScript
- Node.js + Express
- MongoDB
- Chart.js

## How it works
Users log activities like travel, food, and electricity use. The system calculates CO2 emissions and shows results in charts.

## Purpose
To help users become more aware of their carbon footprint and reduce emissions.
